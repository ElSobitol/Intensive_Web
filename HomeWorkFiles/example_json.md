**Задача №2:** проверить JSON на Well formed:

```json
{

        ""surname"": ""Иванов""

        ""name"": ""Иван""

        ""patronymic"": ""Иванович""

        ""birthdate"": ""01.01.1990""

        ""birthplace"": ""Москва""

        ""phone"": ""8 926 766 48 48""

}
```
В данном коде слеюующие ошибки:
1. двойные скобки
2. нет запятых после аргументов

Код должен выглядеть так

{

        "surname": "Иванов",

        "name": "Иван",

        "patronymic": "Иванович",

        "birthdate": "01.01.1990",

        "birthplace": "Москва",

        "phone": "8 926 766 48 48"

}
